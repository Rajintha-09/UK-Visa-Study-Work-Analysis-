
## Project Overview
This project presents a data-driven analysis of UK Study and Work Visa applications using 
historical Managed Migration datasets published by the UK Home Office.

The analysis focuses on application trends, nationality patterns, visa outcomes, industry 
distribution, and extensions of stay, covering the period:
- Study Visas: 2010 – 2024
- Work Visas: 2010 – 2023

## Dataset Sources
- UK Home Office – Managed Migration (Historic Data)
- Source: Data.gov.uk

## Technologies Used
- SQL Server (Data storage, cleaning, transformation)
- Power BI (Data modeling, DAX, visualization)
- Excel (Raw datasets)
- GitHub (Version control and project documentation)

## Key Insights
- Study visa applications dominate overall UK visa demand
- China and India are the top nationalities across study and work visas
- Higher Education Institutions attract the majority of international students
- Work visa demand is highest in Health, IT, Professional Services, and Education sectors
- Extensions of stay increased significantly after 2021
